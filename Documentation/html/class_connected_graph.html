Detailed Description

Connected graph class.

A class that represents a graph, which each node is an airport Each edge has the flight information Node: airport -> all the airport information adjacent -> all the flights from the current airport km -> distance in real units(km) currentFlight -> current flight from the airport distance -> distance in nodes
